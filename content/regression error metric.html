regression error metric

1.error metric

  • MAE 平均絕對誤差 :對真實值-預測值(誤差)取絕對值後(黃色直線距離)相加，再平均。

    MAE=1n∑i=1n∣真實值i−預測值i∣\text{MAE} = \frac{1}{n} \sum_{i=1}^{n} \left| \text{真實值}_i - \text{預測值}_i \right|

  • MSE 均方誤差 :對誤差平方(正方形面積)相加後再平均。

    MSE=1n∑i=1n(真實值i−預測值i)2\text{MSE} = \frac{1}{n} \sum_{i=1}^{n} \left( \text{真實值}_i - \text{預測值}_i \right)^2

           

  • RMSE 均方根誤差:對誤差平方(正方形面積)相加後，平均再開根號。

    RMSE=1n∑i=1n(真實值i−預測值i)2

  • MPE平均百分比誤差:(實際值-預測值)/實際值*100%後取平均。

    MPE=1n∑i=1n真實值i−預測值i真實值i×100%\text{MPE} = \frac{1}{n} \sum_{i=1}^{n} \frac{\text{真實值}_i - \text{預測值}_i}{\text{真實值}_i} \times 100\%

  • MAPE 平均絕對百分比誤差 :abs((實際值-預測值)/實際值*100%後取平均

    MAPE=1n∑i=1n∣真實值i−預測值i真實值i∣×100%\text{MAPE} = \frac{1}{n} \sum_{i=1}^{n} \left| \frac{\text{真實值}_i - \text{預測值}_i}{\text{真實值}_i} \right| \times 100\%

2.goodness of fit

  • R²決定係數
符號 全名 公式 意思
SST Total Sum of Squares sum((y - mean(y)).^2) y 的總變異
SSR Regression Sum of Squares sum((y_pred - mean(y)).^2) 模型解釋到的變異
SSE Error Sum of Squares sum((y - y_pred).^2) 模型沒解釋到的變異（殘差）

*mean(y)為原始數據取平均， y為某一點的原始數據, y_pred為某一點預測方程式的值。

SST=SSR+SSE;

R2=1−∑(某一點的原始數據−某一點預測方程式的值)2∑(某一點的原始數據−原始數據取平均)2=1−殘差平方和 (SSR)總平方和 (SST)R^2 = 1 - \frac{\sum \left( \text{某一點的原始數據} - \text{某一點預測方程式的值} \right)^2}{\sum \left( \text{某一點的原始數據} - \text{原始數據取平均} \right)^2} = 1 - \frac{\text{殘差平方和 (SSR)}}{\text{總平方和 (SST)}}

Q:數值多少算好

A:

指標 單位 越小越好？ 對大誤差（outlier）的敏感度 什麼情況下「看起來小」才算好？ 常見應用場景建議
MAE 跟 y 相同單位 是 低（robust） 跟 y 的平均值或範圍比，小於 5–15% 左右常被接受 業務容易理解、對大錯不特別懲罰
MSE y 單位的平方 是 極高 很難直接看，通常只跟其他模型比 訓練時用（可微分）
RMSE 跟 y 相同單位 是 高 常比 MAE 大一點，跟 y 平均值比，小於 10–20% 常被接受 最常用（平衡解釋力與懲罰大錯）
MAPE 百分比 (%) 是 中等（但 y 接近 0 會爆炸） <10% 優秀、10–20% 好、20–50% 可接受、>50% 通常偏差 銷售、需求預測（規模無關）
MPE 百分比 (%) 接近 0 最好 中等（有正負） 接近 0（有系統性偏誤時會遠離 0） 檢查是否有 bias

Q:MAE RMSE 數值會差不多嗎?

A:

情況一：誤差都差不多大 err = [1, 1, 1, 1, 1] MAE = 1.0 RMSE = 1.0 ← 幾乎一樣

情況二：有一個很大的離群誤差 err = [1, 1, 1, 1, 10] MAE = 2.8 RMSE = 4.6 ← 差很多，因為 10² = 100 被放大了

Q:R² 決定係數 — 多少算好？

A:

  • R² > 0.9–0.95：優秀（物理模型強的領域）
  • R² 0.8–0.9：良好（大多數工程應用）
  • R² 0.6–0.8：可接受（現場數據噪音大）
  • R² < 0.6：通常不夠好，除非是極端複雜系統
